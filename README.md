# WildGuard AI

WildGuard AI is an AI-powered wildlife monitoring and detection platform developed using YOLO and Streamlit. The system analyzes wildlife videos to detect and classify animal species, estimate wildlife activity levels, identify rare species, and generate analytical reports for ecological monitoring.

The platform includes real-time detection, activity scoring, species analytics, historical records, and report generation features, providing a complete solution for intelligent wildlife surveillance and conservation support.

## Features

* YOLO-based wildlife species detection from videos
* Wildlife activity scoring and risk analysis
* Rare species identification and alerts
* Interactive analytics dashboard
* Historical detection records
* Automated report generation
* MySQL database integration
* Modern Streamlit-based user interface

## Technologies Used

* Python
* YOLO (Ultralytics)
* Streamlit
* OpenCV
* MySQL
* Pandas
* Plotly
